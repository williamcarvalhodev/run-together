# RunTogether

RunTogether is a responsive website designed to help people find local running
groups, weekly training sessions and upcoming running events.

## Project Goal

The goal of this project is to make running more social, accessible and
motivating by centralising information about running groups and events.

## Pages

- Home
- Running Groups
- Events
- About & Contact

## Technologies

- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons
- Git and GitHub

## Main Features

- Responsive Bootstrap navigation
- Mobile-first layout
- Running group cards
- Running events table
- Email hyperlink
- External links
- Accessible semantic HTML
