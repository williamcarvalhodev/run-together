# RunTogether

RunTogether is a responsive four-page website designed to help people discover local running groups, weekly training sessions and upcoming running events.
The project was created as part of the UCD Professional Academy Web Design Assignment.

## Project Overview

Many people are interested in running but find it difficult to stay motivated when training alone.
They may also struggle to find local running groups that match their pace, preferred distance and experience level. Information about running communities and events is often spread across different websites and social media platforms.

RunTogether provides a central and user-friendly website where visitors can find suitable running groups, compare training information and explore upcoming running events.

## Website Goal

The goal of the RunTogether website is to make running more social, accessible and motivating.
The website helps visitors:

- Find local running groups.
- Compare pace, distance and experience levels.
- Check weekly training sessions.
- Discover upcoming running events.
- Access useful information for beginner runners.
- Contact the running community.

## Target Audience

The website is designed for:

- Beginner and intermediate runners.
- People who prefer running with a group.
- Runners looking for motivation and consistency.
- People searching for local running events.
- Individuals who are new to the local area.
- Runners preparing for 5K, 10K, half marathon or marathon events.

## Website Pages

### Home

The Home page introduces the RunTogether concept and includes:

- Hero banner.
- Main website message.
- Benefits of running with a group.
- Featured running groups.
- Beginner running tips.
- Embedded YouTube video.
- Footer with contact information and external links.

### Running Groups

The Running Groups page allows visitors to explore groups based on pace, distance and experience level.
It includes:

- Running group cards.
- Group level and distance information.
- Weekly training details.
- Meeting locations.
- Registration forms.
- A section for suggesting a new running group.

### Events

The Events page presents upcoming running events in a clear and organised format.
It includes:

- Event introduction banner.
- Event benefits.
- Responsive event table.
- Event dates, distances and locations.
- External event links.
- Running events image gallery.

### Contact

The Contact page provides information for visitors who want to get in touch.
It includes:

- Contact banner.
- Contact information.
- Email hyperlink.
- Contact form.
- Google Maps location.
- Useful external links.
